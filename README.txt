CAPSTONE PROJECT

Lanterna's version : 2.1.9 

Am Anfang kann Nutzer den Schwriegkeitsgrad wählen. Diese Levels unterscheiden sich durch die Größe des Labyrinths, die Anzahl von Monsters und die Density. Nach dem Wählen wird zufällig ein Labyrinth, entsprechend des Levels, erzeugt.

Dem Spieler steht 3 Lebens zur Verfügung. Er muss insgesamt 5 Schatzkammern einsammeln, um den Ausgang aufmachen und das Spiel gewinnen zu können.

Jede Zeit kann der Nutzer Esc aufrufen. Dann gibt es ein Spielmenu, auf dem er 

Continue
Save game 1
Save game 2
Load game 1
Load game 2
Legends
Exit

auswählen kann.

Zur Navigating des Pacman benutzt man natürlich Arrow up, Arrow down, Arrow left, Arrow right.

Die Main Methode befindet sich in dem Datei Main.java.
